# Application Design Documents

Design documents for the Physical AI Studio **application** — the FastAPI backend (`application/backend`) and React UI (`application/ui`).

> No application design docs have been merged yet. This directory follows the upstream convention (`application/docs/designs/`) and is where app-side designs such as remote-training, trainer-service, and backend abstractions will live when their PRs land.

When adding an application design document:

1. Place the file under `application/docs/designs/` (e.g. `application/docs/designs/remote-training.md`).
2. Link it from `docs/design/README.md` under **Application Designs** so it is discoverable from the master index.
3. Open a PR to `upstream/main` for team review before merging.
